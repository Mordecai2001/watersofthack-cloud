# WaterSoftHack 2026 — Cloud / Edge / Hybrid Workshops

Hands-on notebooks for the three-day Cloud / Edge / Hybrid track at
[WaterSoftHack 2026](https://watersofthack.github.io/), run by Mihail Neagu (University of Iowa).

These are the **student (fill-in) versions**. You run them on the shared JupyterHub, no local install.

| Day | Notebook | Topic |
|-----|----------|-------|
| Tue Jul 21 | `Day1_Cloud.ipynb` | Cloud computing: pull live USGS data, train a streamflow now-cast, scale it out |
| Wed Jul 22 | `Day2_Edge.ipynb` | Edge computing: simulate smart stream gauges under bandwidth and power limits |
| Thu Jul 23 | `Day3_Hybrid.ipynb` | Hybrid: edge triage plus cloud forecasting in one flood early-warning pipeline |

## How to open a notebook on the hub

Click the day's link (your instructor will share it). It logs you in and drops that day's
notebook straight into your workspace. Re-clicking pulls the latest version without erasing your work.

The scenario runs across all three days: a real-time flood early-warning system built from a
network of USGS stream gauges.
